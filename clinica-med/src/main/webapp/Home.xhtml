Frameworks de desenvolvimento

PrimeFaces é um framework open source popular para JavaServer Faces com mais de 100 componentes, toque mobilekit otimizado, quadro empurrão, a validação do lado do cliente, motor do tema e muito mais.

Hibernate ORM permite que os desenvolvedores a escrever mais facilmente aplicações cujos dados sobrevive ao processo de candidatura. Como um quadro Objeto / Relational Mapping (ORM), Hibernate está preocupado com a persistência de dados como se aplica a bancos de dados relacionais (via JDBC).
